# Телеграм-бот для приема заказов

Бот на aiogram для рекламы и приема заказов на разработку телеграм-ботов.

## Установка и запуск

### 1. Клонирование и настройка

1. Создайте файл `.env` на основе `env.example`:
```bash
BOT_TOKEN=your_bot_token_here
ADMIN_ID=your_telegram_user_id_here
```

2. Получите токен бота у [@BotFather](https://t.me/BotFather)
3. Узнайте свой Telegram ID (можно через [@userinfobot](https://t.me/userinfobot))

### 2. Структура папок для фото

Создайте следующие папки и добавьте в них фотографии:

```
data/
├── start/          # Фото при старте бота
├── about/          # Фото в разделе "О нас"
└── bot_types/
    ├── anketnik/   # Фото для типа "Анкетник"
    ├── control/    # Фото для типа "Контроль сотрудников"
    ├── vizitka/    # Фото для типа "Визитка"
    └── custom/     # Фото для типа "Индивидуальный заказ"
```

### 3. Запуск через Docker

```bash
docker-compose up -d
```

### 4. Запуск без Docker

```bash
pip install -r requirements.txt
python bot.py
```

## Функционал

- Главное меню с разделами: "Типы ботов", "О нас", "Заказать"
- Просмотр типов ботов с описанием и фотографиями
- Процесс заказа с сбором данных (имя, контакт, тип бота)
- Автоматическая отправка заказов администратору в личку
- Все кнопки инлайновые
- На любое сообщение показывается главное меню
